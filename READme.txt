定位器项目：

开发流程:
	1.common公共层：更改底层printf逻辑打印到串口，利用file，line等关键字定义自己的debugprintf方式：打印当前文件及行数，VA_ARGS_添加可变参数
，定义自己的延时函数，创建总数据结构体（包含芯片uid，经纬度及方向，定位时间，计步信息，最终上传的数据及长度），搬运cjson源文件来处理数据
	2.driver底层驱动层：主要完成各芯片与主控之间的底层链接，通过cubemx配置
	3.interface硬件接口层：主要完成1.AT6558R芯片：封装了发命令函数，读gps定位数据函数，低功耗相关函数
								2.DS3553芯片：封装了读写寄存器函数，初始化调教芯片函数，读计步芯片函数
								3.QS100芯片：封装了唤醒，软重启，发命令，芯片初始化，查询附着，创建客户端，链接服务端，关闭客户端，底层发数据，总发数据函数，低功耗相关函数
								
	4.application应用层：实现了低功耗逻辑：gps，互联，主控的低功耗；实现了获取gps，计步信息有效数据并传入上传数据struct的功能，转换为josn数据格式的功能
